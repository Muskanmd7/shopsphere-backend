# 🛒 ShopSphere Backend

A scalable e-commerce backend built using Spring Boot with secure JWT authentication and role-based authorization.

## 🚀 Features

- User Registration & Login
- JWT Authentication
- Role-Based Authorization (ADMIN / USER)
- BCrypt Password Encryption
- Spring Security
- Stateless Authentication
- RESTful APIs
- PostgreSQL Database
- Product CRUD Operations
- Secure API Endpoints

## 🛠️ Tech Stack

- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- JWT (JJWT)
- PostgreSQL
- Maven
- Hibernate
